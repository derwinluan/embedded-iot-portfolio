# Test Guide — #05 UNO UART MQTT Bridge

        ## Reference
        Product metrics: sort **5** in the APP product catalog (generated by `hardware/generate_app_product_excel.py`, sheet *Product Metrics Overview*).

        ## Steps
        1. Open `firmware/` in Arduino IDE or PlatformIO
2. Select the correct board and flash
3. Open Serial Monitor at 9600 or 115200 baud
4. Flash `uno_slave.ino` on UNO and `gateway.ino` on ESP32; cross-connect UART

        ## Expected
        - Serial Monitor shows JSON telemetry or command echo
        - Without hardware, compile-only syntax check is acceptable
