# Test Guide — #27 ESP32 CAN Parser Library

        ## Reference
        Product metrics: sort **27** in the APP product catalog (generated by `hardware/generate_app_product_excel.py`, sheet *Product Metrics Overview*).

        ## Steps
        1. Open `firmware/` in Arduino IDE or PlatformIO
2. Select the correct board and flash
3. Open Serial Monitor at 9600 or 115200 baud
4. Build `main.cpp` with PlatformIO; verify CAN JSON on serial

        ## Expected
        - Serial Monitor shows JSON telemetry or command echo
        - Without hardware, compile-only syntax check is acceptable
